# Minecraft UUID Tracker Bot 🤖⛏️

A Discord bot designed to solve the problem of tracking Minecraft players who frequently change their in-game names. It uses the official Mojang API to fetch and log historical name data based on static player UUIDs.

## How it Works
* Stores a database of registered player UUIDs.
* Queries the Mojang API to verify current names against stored historical data.
* Logs any detected name changes directly into a Discord channel with timestamps.
